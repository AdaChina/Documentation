# 错误代码参照表

适用API:
- 班牌端 V1
- 后台管理 V1

具体错误信息，请参照错误返回的```message```
```json
{
  "error_code": 1003,
  "message": "缺少ID参数"
}
```

| 错误码 error_code | 错误描述 | HTTP状态码 |
| -- | -- | -- |
| 1001 | 未找到资源 (通用类错误) | 404 |
| 1002 | 资源无法处理 (通用类错误) | 422 |
| 1003 | 缺少参数 (通用类错误) | 400 |
| 1004 | 参数不正确（通用类错误） | 422 |
| 1005 | 第三方服务错误 | 422 |
| 1006 | 设备不存在 | 404 |
