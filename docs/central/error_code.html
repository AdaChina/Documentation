  "error_code": 1003,
  "message": "缺少ID参数"
}
错误码 error_code错误描述HTTP状态码

1002资源无法处理 (通用类错误)422
1003缺少参数 (通用类错误)400
1004参数不正确（通用类错误）422
1005第三方服务错误422
1006设备不存在404
