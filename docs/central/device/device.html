# 设备信息

# 获取基础信息

获取设备序列号，系统归属信息，以及IoT平台信息。

GET /device/base

响应字段

字段名描述
serial设备序列号
iot_id设备IoT平台ID
iot_secret设备IoT平台密钥
system: name所属系统名称
system: api_endpoint所属系统API地址
system: short_code所属系统短码

响应示例

成功响应:

Status: 200 OK

{
  "serial": "AC123456789",
  "iot_id": "98575177CA",
  "iot_secret": "K#$vOAOO31",
  "system": {
    "name": '智慧校园',
    "api_endpoint": "https://classbrand.staging.adachina.net",
    "short_code": "class-brand"
  }
}
