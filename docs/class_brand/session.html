# 身份验证API

登陆API管理班牌后台页面登陆和登出动作

# 登陆

POST /sessions

请求参数

参数名参数类型必填描述示例
emailstring是邮箱地址user@school.edu
passwordstring是密码password
remember_mestring否持久登陆1

响应字段

字段名描述
id用户ID
type用户类型
name用户姓名
avatar_url用户头像
type 参数对照描述
SystemAdmin系统管理员
SchoolAdmin学校管理员

响应示例

成功响应:

Status: 200 OK

{
  "id": 1,
  "type": "SystemAdmin",
  "name": "王毅",
  "avatar_url": "https://image.com/avatar.jpg"
}

失败响应:

Status: 401 Unauthorized

{
    "base": "邮箱和密码组合不正确"
}

# 登出

DELETE /sessions

响应示例

成功响应:

Status: 302 Found

登出成功后，会重定向到登陆页面。
