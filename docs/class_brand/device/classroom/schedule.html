# 获取每日课程表

提供节次，每日节次课程信息，以及课程考勤信息。默认返回当天及前后3天，共7天的课程表。

GET ${API_ENDPOINT}/classroom/schedules/daily

请求参数

参数名参数类型必填描述示例
from_dateinteger否开始日期2000-01-20
to_datestring否结束日期2000-02-01
